# Diabetic Retinopathy Detection

A deep learning-based application for detecting diabetic retinopathy from retinal fundus images using Swin Transformer architecture.

## Features
- Detects diabetic retinopathy from retinal scans
- Supports Grad-CAM and Eigen-CAM visualizations
- Includes evaluation metrics and confusion matrix plotting
- Training and testing pipeline included

## Tech Stack
- Python
- PyTorch
- OpenCV
- Swin Transformer
- Deep Learning / Computer Vision

## Project Structure
- `train.py` – Model training
- `test.py` – Testing script
- `evaluate.py` – Performance evaluation
- `gradcam.py` – Heatmap visualization
- `app.py` – Main application file

## Results
Achieved accurate retinal image classification using transformer-based architecture for medical image analysis.
